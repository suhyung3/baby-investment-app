# 🍼 우리 아기 재테크 계산기

지금 증여한 돈이 20년 후 얼마가 될까요? 복리의 마법을 직접 확인해보세요!

## 기능

- 초기 증여금 및 월 적립금 설정
- 연 기대 수익률 계산
- 10년 단위 2차 증여 전략 시뮬레이션
- 자산 성장 그래프 시각화
- 연도별 상세 자산 변화 테이블

## 로컬 개발

```bash
npm install
npm start
```

## 배포

Netlify, Vercel 등의 플랫폼에서 자동 배포 가능합니다.

## 기술 스택

- React 18
- Recharts (차트 라이브러리)
- Tailwind CSS
